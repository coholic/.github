## 📕 방식
- 매 주 1회 알고리즘 비대면 + CS 대면 스터디 진행
- 알고리즘은 [백준](https://www.acmicpc.net/), [프로그래머스](https://programmers.co.kr/) 등으로 진행
- CS는 Notion에 정리한 후, 예상 질문 & 답변 만들기 및 개념 설명
- 최종 목표는 Notion에 정리한 내용을 완벽하게 숙지한 뒤, 이를 바탕으로 블로그 포스팅
